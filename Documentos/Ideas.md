Se proponen las siguientes ideas de aplicación para el proyecto:

## [Idea 1: PassGo (Sleccionada)](Documentos/Ideas.md)

PassGo es una aplicación la cual almacena y genera contraseñas, lo anterior se debe a la gran vulnerabilidad que se presenta en diferentes sitios, de esta manera los usuarios pueden tener una contraseña diferente para cada sitio o aplicación.

### [Aplicaciones similares](Documentos/Ideas.md)

✔️ [Gestor de contraseñas Google](https://play.google.com/store/apps/details?id=com.google.android.apps.credentialmanager)    
✔️ [Kaspersky Password Manager](https://galaxystore.samsung.com/detail/com.kaspersky.passwordmanager)  

## [Idea 2: Control de Consumo de Agua](Documentos/Ideas.md)
